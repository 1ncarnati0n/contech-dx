## 4. 수식 분석

### 4.1 수식 개요
- **총 수식 개수**: 3개
- **시트간 참조**: 없음 (모두 같은 시트 내 참조)

### 4.2 수식 상세

#### 4.2.1 D3: 공사기간 표시
```
=M33
```
- **의미**: M33 셀의 값(개월 단위 공사 기간)을 표시
- **용도**: 공사기간을 헤더 영역에 표시

#### 4.2.2 M32: 작업일수 합계
```
=SUM(L7:M31)
```
- **의미**: L7부터 M31까지의 범위 합계
- **참고**: M7~M31은 비어있으므로 실제로는 L7~L31의 합계
- **용도**: 모든 층/구역의 작업일수 합계 계산

#### 4.2.3 M33: 개월 환산
```
=ROUNDUP((15+M32+(M32/6))/30,2)
```
- **의미**: 
  - 기본 15일 + 작업일수 합계(M32) + 여유일수(M32/6)를 30으로 나누어 개월 단위로 환산
  - ROUNDUP 함수로 소수점 둘째 자리까지 올림
- **계산 로직**:
  1. 기본 여유일수: 15일
  2. 총 작업일수: M32
  3. 추가 여유일수: M32/6 (작업일수의 1/6)
  4. 총 일수 = 15 + M32 + (M32/6)
  5. 개월 환산 = 총 일수 / 30
  6. 소수점 둘째 자리까지 올림
- **용도**: 총 공사 기간을 개월 단위로 계산
