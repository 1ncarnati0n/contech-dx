# Next.js 16 + Supabase Auth Skill

Production-ready Next.js 16 + Supabase authentication and role-based access control system.

## What's Included

### SKILL.md
Main skill documentation with:
- Complete implementation workflow
- Next.js 16 compatibility guide
- Role-based permission system
- Troubleshooting section
- Quick reference

### scripts/
- `setup-admin.sql` - Promote first user to admin
- `verify-setup.sql` - Diagnostic queries

### references/
- `database-schema.md` - Complete SQL schema with explanations
- `file-structure.md` - Project structure guide
- `troubleshooting.md` - Common issues and solutions

### assets/
- `templates/` - Copy-paste ready code templates
  - Supabase clients (browser, server, middleware)
  - Permission system (server and client)
  - Auth components (login, signup)
  - TypeScript types
- `examples/` - Real-world implementation patterns
  - Server actions
  - Protected pages
  - Client components

## Quick Start

1. Read SKILL.md for complete implementation guide
2. Copy templates to your project
3. Run database schema
4. Promote first admin
5. Start building!

## Features

- 4-tier role system (admin, main_user, vip_user, user)
- Row Level Security (RLS) policies
- Next.js 16 full compatibility
- Server and client permission checks
- Real-time authentication state
- OAuth support ready
- Production battle-tested patterns

## Requirements

- Next.js 16+
- Supabase account
- @supabase/ssr package
- TypeScript
